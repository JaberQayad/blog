### 3. Create a new page

To create a new page, you can create a new markdown file in the `content/page` directory. The file should have the following front matter:

```yaml
---
title: "New Page"
date: 2020-07-01T00:00:00-04:00
draft: false
---

This is the content of the new page.
```
