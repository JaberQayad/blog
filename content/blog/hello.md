### 4. Create a new post

To create a new post, you can create a new markdown file in the `content/post` directory. The file should have the following front matter:

```yaml

---
title: "New Post"
date: 2020-07-01T00:00:00-04:00
draft: false

---


# This is the content of the new post.
```
